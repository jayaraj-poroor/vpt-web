Sensoid

Sensoid is a cloud platform for executing streaming queries on sensor data.

Sensoid integrates gateway devices, smart phones, and other Internet-connected on-site devices as query-capable IoT edge nodes.

Sensoid queries pack power let's see how

Your queries can contain embedded actions.

Your queries are computed on IoT edge devices such as on-site gateway devices or smart phones.

Embedded actions? raise an alarm, drive an actuator,post to a URL, send a tweet, and many more.

With edge computing, embedded actions, and other power-packed features imagination is the limit!

Dynamic, reprogrammable sensor data streams
Sensor data streams are defined by streaming queries which can be created, modified, removed dynamically using our API.

Smart disconnected operation
Sensor events can be locally responded to even when there is disruption in Internet connectivity.
To minimize information loss in case of longer disconnects, application-specific priorities can be attached to queries for deciding what data to keep and what to discard.

Bandwidth optimization
Queries filter relevant information in an application-specific manner and transmit only the relevant information to our back-end cloud.

Local real-time responses
Quick local real-time responses can be made to sensor events without requiring Internet/back-end cloud always to be in the event-response loop.

To summarize

With Dynamic reprogrammable streams, bandwidth/resource optimization, local real-time responses, smart disconnected operation, and simple SQL-like syntax, Sensoid gives you power & simplicity.
